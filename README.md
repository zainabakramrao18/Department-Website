# University Website

A simple static website for a university showcasing basic pages.

How to view
- Open `index.html` in your web browser (double-click or use "Open File").

Included files
- `index.html` — homepage
- `about.html` — about page
- `faculty.html` — faculty listing
- `labs.html` — labs information
- `contact.html` — contact page

Notes
- This is a static site (HTML files only). No build step required.
